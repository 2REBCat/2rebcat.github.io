# 2RebCat Hugo Site

This repository contains the Hugo source for the portfolio site.

## Structure

- `content/`: site content and front matter
- `themes/2rebcat-academic/`: custom Hugo theme for the site
- `.github/workflows/hugo.yml`: GitHub Pages deployment workflow

## Local development

1. Install Hugo Extended.
2. Run `hugo server` in the repository root.
3. Open the local address shown by Hugo.

## Deployment

Push to `main` and GitHub Actions will build the site and deploy it to GitHub Pages.
